# bench/results/

Committed, unedited output from `bench/collect_environment.sh` and
`bench/run_replay_benchmark.sh`, captured on the project's actual target
machine (see `bench/ENVIRONMENT.md`) — never retyped or summarized, so
every number cited elsewhere in the repo traces back to a command that
was actually run on real hardware.

Naming convention: `environment_YYYYMMDD.txt`,
`replay_baseline_YYYYMMDD_<ticker>.txt`.

Empty until that first real run happens — see `bench/ENVIRONMENT.md` for
why nothing here yet is a "smoke test on the wrong machine" caveat.
